## Task 1: Working with Files

```bash

# View the file

# Remove the file
rm test.txt
```

## Task 2: Environment Variables

```bash
# Check environment variables
env | grep PATH

# Set a variable
export MY_VAR="test"
echo $MY_VAR
```
