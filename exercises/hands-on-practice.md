# Hands-on Practice Exercise

This exercise provides hands-on practice with the concepts covered in the training.

## Overview

This exercise will guide you through practical implementation of the concepts learned.

## Task 1: Setup

1. Navigate to the project directory
2. Review the configuration files
3. Understand the project structure

## Task 2: Implementation

Complete the following tasks:

### Sub-task 2.1

```bash
# Run the setup script
./scripts/setup.sh
```

### Sub-task 2.2

Modify the configuration as needed:

```yaml
# Example configuration
setting: value
```

## Task 3: Validation

Verify your implementation:

1. Check the output
2. Review logs
3. Test functionality

## Challenge

> **Challenge:** Can you extend this exercise to include additional features?

## Summary

In this exercise, you have:
- Completed setup tasks
- Implemented core functionality
- Validated your work
